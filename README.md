# Prokaryotic adaptation to high temperature

Understanding the drivers of high temperature adaptation in archaea and bacteria.

## Repository structure

- `Rmds/` – R Markdown reports (01_fetch-raw-data, 02_integrate-metadata, etc.).
- `analysis/functions/` – Reusable R functions for data and models.
- `analysis/utils/` – Helper utilities (plot themes, small helpers).
- `data/raw/` – Raw input data (downloaded from Tempura, BioCyc, NCBI, KEGG, GTDB). Treat as read-only. [web:24][web:28]
- `data/processed/` – Cleaned/merged datasets created by the Rmds (can be regenerated).
- `data/external/` – External reference files (e.g., full GTDB trees, mapping tables).
- `results/figures/` – Plots and figures generated by analyses.
- `results/models/` – Saved model objects, summaries, and diagnostics. [web:19][web:32]
